# Kevin Ma — Personal Homepage

Single-page research portfolio for Kevin Ma (foundation model robotics, research & development).

- `index.html` — the entire site (self-contained HTML + inline CSS/JS).
- `assets/` — project demo media (VLA results GIF, Qwen2-VL quantization chart).

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy (GitHub Pages)

Pushed to a `*.github.io` user-site repo, it serves automatically at
`https://kevinma0215.github.io`.
